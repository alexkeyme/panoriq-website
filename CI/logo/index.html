Which size goes where?

16, 32, 48Browser favicon (use favicon/*.png)
24, 32, 48Toolbar / in-UI icons, small list avatars
56Android Material FAB (floating action button)
64, 72, 96Large buttons, cards, sidebar branding
128, 144macOS / desktop app icons
180iOS home-screen (Apple touch icon)
192Android home-screen / PWA manifest
256, 512App Store listing, social profile, Slack workspace
1024, 2048Print / high-DPI / master for any future resize
Lockup 400wEmail signatures, compact nav bars
Lockup 800wRetina nav, docs headers, presentations
Lockup 1600wWebsite hero, marketing pages
